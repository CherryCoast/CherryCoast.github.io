一、什么是cookie(储存在用户本地终端上的数据)

  Cookie，有时也用其复数形式 Cookies。类型为“小型文本文件”，是某些网站为了辨别用户身份，进行Session跟踪而储存在用户本地终端上的数据（通常经过加密），由用户客户端计算机暂时或永久保存的信息

二、cookie的使用(java版)

1、cookie的创建

  （1）不设置有效期(临时存储):存储在浏览器的运行内存中,浏览器关闭则cookie失效

//获取请求对象
HttpServletRequest request = ServletActionContext.getRequest();
//获取响应对象
HttpServletResponse response = ServletActionContext.getResponse();

//获取前端的用户名和密码
String name = request.getParameter("name");
String pwd = request.getParameter("pwd");

//将用户名和密码存放到cookie中
Cookie cookie1 = new Cookie("username", name);
Cookie cookie2 = new Cookie("password", pwd);

//响应到浏览器
response.addCookie(cookie1);
response.addCookie(cookie2);

注意以下几点

1.一个cookie对象只能存储一条数据,多条数据需要新建多个cookie对象

2.如果没有设置cookie的有效期的话，在浏览器关闭的同时cookie也随之消失

  （2）设置有效期(定时存储):存储在客户端的硬盘中,在有效期符合路径要求的请求都会携带该信息 
HttpServletRequest request = ServletActionContext.getRequest();
HttpServletResponse response = ServletActionContext.getResponse();
String name = request.getParameter("name");
String pwd = request.getParameter("pwd");
Cookie cookie1 = new Cookie("username", name);
Cookie cookie2 = new Cookie("password", pwd);
//给用户名设置有效期（注意值是义秒为单位的）
cookie1.setMaxAge(5*24*3600);
response.addCookie(cookie1);
response.addCookie(cookie2);
return super.execute();

  （3）设置有效路径

HttpServletRequest request = ServletActionContext.getRequest();
HttpServletResponse response = ServletActionContext.getResponse();
String name = request.getParameter("name");
String pwd = request.getParameter("pwd");
Cookie cookie1 = new Cookie("username", name);
Cookie cookie2 = new Cookie("password", pwd);
//设置有效路径
cookie1.setPath("/user/cookie");
//给用户名设置有效期（注意值是义秒为单位的）
cookie1.setMaxAge(5*24*3600);
response.addCookie(cookie1);
response.addCookie(cookie2);
return super.execute();

2、cookie的获取

HttpServletRequest request = ServletActionContext.getRequest();
//获取所有的cookie
Cookie[] cookies = request.getCookies();
//获取的cookie不为空
if(cookies != null) {
    //遍历所有的cookie
    for (Cookie cookie : cookies) {
        //输出对应的key和value
        System.out.println("key : " + cookie.getName() + "  value : " + cookie.getValue());
    }
}
return super.execute();

一、lambda表达式简介

1、什么是lambda表达式

  lambda表达式是JDK1.8添加的一个新特性，lambda把一个匿名函数进行了简化操做，

2、为什么需要lambda表达式

  lambda表达式可以将匿名函数进行非常简洁的实现

例如：

package com.blog.lambda;

public class LambdaDemo01 {
    //使用接口实现类
    Comparetor com01 = new ImpComparetor();
    //使用匿名内部类
    Comparetor com02 = new ImpComparetor(){
        @Override
        public int compare(int num1, int num2) {
            return num1 - num2;
        }
    };
    //使用lambda表达式
    Comparetor com03 = (num1, num2) -> {
        return num1 - num2;
    };
    //在方法中只有一句时可以省略大括号
    Comparetor com04 = (num1, num2) -> num1 - num2;

}

class ImpComparetor implements Comparetor {
    @Override
    public int compare(int num1, int num2) {
        return num1 - num2;
    }
}

interface Comparetor {
    int compare(int num1, int num2);
}

3、lambda表达式对接口的要求

  虽然有些时候可以使用lambda表达式来对接口进行简单的实现，但并不是所有的接口都能用lambda表达式来实现，如果接口中所定义的方法超过了一个，则不能使用lambda表达式

  Java1.8对接口加了一个新的特征：default——在接口中定义非抽象方法。通俗点讲default就是给接口赋予了原来抽象类的能力，实现类可以像使用抽象类的方法一样，直接使用接口里的方法。

  @FunctionalInterface:修改函数式接口，保证接口中的抽象方法只有一个

二、lambda的基本语法

接口代码：

package com.blog.lambda;

public class LambdaDemo01 {
    public static void main(String[] args) {}
}

@FunctionalInterface
interface Test01 {
    void test();
}

@FunctionalInterface
interface Test02 {
    void test(String name);
}

@FunctionalInterface
interface Test03 {
    void test(String name, String str);
}

@FunctionalInterface
interface Test04 {
    int test(String name);
}

@FunctionalInterface
interface Test05 {
    int test(String name, String str);
}
lambda表达式的基础语法，可以将lambda表达式看做是一个匿名函数

参数列表() 方法体{}

():用来描述参数列表
{}:用来描述方法体
->:lambda运算符，读作goes to

1、test01:

  1. 无参，只需要用()表示
  2. 无返回值 方法体内只有一条语句，可以省略大括号
    Test01 test01 = () -> System.out.println("这是test01接口");
test01.test();

    2、test02:

  3. 只有一个参数：可以省略括号，还可以省略参数类型，因为接口已经定义了参数类型和数量
  4. 无返回值 方法体内只有一条语句，可以省略大括号
    Test02 test02 = name -> System.out.println(name + "调用了test02接口");
test02.test("张三");

    3、test03:

  5. 有两个参数（不可以省略括号），
  6. 无返回值 方法体内只有一条语句，可以省略大括号
  7. 注意:如果要省略参数类型就将所有的参数类型都省略，不能只省略一个参数类型
    Test03 test03 = (name, str) -> System.out.println(name + str);
test03.test("张三", "调用了test03接口");

    4、test04:

  8. 只有有一个参数（可以省略括号），
  9. 有返回值 方法体内只有一条语句，可以省略大括号
  10. 注意：如果这一条语句是返回值，那么在省略大括之前必需将return也去掉
    Test04 test04 = name -> name.length();
test04.test("张三");

    5、test05:

  11. 有两个参数（不可以省略括号），
  12. 有返回值 方法体内有两条语句，不可以省略大括号
    Test05 test05 = (name, str) -> {
 System.out.println(name + str);
 return name.length();
};
test05.test("张三","调用了test05接口");

    三、lambda语法进阶

    1、方法的引用

  13. 概述：可以快速的将一个lambda表达式对接口的实现改成一个已经实现此接口的方法
  14. 代码格式：方法隶属名 :: 方法名

代码示例：

package com.blog.lambda;

public class LambdaDemo01 {
    public static void main(String[] args) {
        //写法一
        Test test01 = num -> test(num);
        System.out.println(test01.test(0));
        //写法二
        Test test02 = LambdaDemo01::test;
        System.out.println(test02.test(0));
    }

    public static int test(int num) {
        for (int i = 0; i < 10; i++) {
            num += 2;
        }
        return num;
    }
}
@FunctionalInterface
interface Test {
    int test(int num);
}

注意:

  1. 方法中的参数类型和参数个数必须和接口中定义的方法一致
  2. 返回值的类型也必须和接口中方法所定义的返回值类型一致

2、构造方法的引用

person类代码

package com.blog.bean;

public class Person {
    private String name;
    private int age;

    public Person() {
        System.out.println("这是person类的无参构造方法");
    }

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("这是person类的有参构造方法");
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}

测试代码:

package com.blog.lambda;
import com.blog.bean.Person;

public class LambdaDemo02 {
    public static void main(String[] args) {
        //原始写法
        ImpPerson01 impPerson01 = () -> new Person();
        impPerson01.getPerson();

        ImpPerson02 impPerson02 = (name, age) -> new Person(name, age);
        impPerson02.getPerson("zs", 12);

        //通过方法调用
        ImpPerson01 impPerson03 = Person::new;
        impPerson03.getPerson();

        ImpPerson02 impPerson04 = Person::new;
        impPerson04.getPerson("zs", 12);
    }
}

interface ImpPerson01{
    Person getPerson();
}

interface ImpPerson02{
    Person getPerson(String name, int age);
}

一、概述

  Java的反射（reflection）机制是指在程序的运行状态中，可以构造任意一个类的对象，可以了解任意一个对象所属的类，可以了解任意一个类的成员变量和方法，可以调用任意一个对象的属性和方法。这种动态获取程序信息以及动态调用对象的功能称为Java语言的反射机制。反射被视为动态语言（程序运行时，允许改变程序结构或变量类型）的关键。

二、class类

1、class结构

  class类是整个反射的源，在jdk1.8的api结构如下

public final class Class<T>
extends Object
implements Serializable, GenericDeclaration, Type, AnnotatedElement

2、取得class类对象的三种方式

1. 使用object提供的返回class类对象的方法

Student student = new Student();
Class clazz = student.getClass();

2. 利用类.class,Hibernate大多使用此方式

Class clazz = Student.class();

3. 使用class类的静态方法forName()取得，此方法抛出ClassNotFoundException异常

try {
    Class clazz = Class.forName("com.blog.Student");
    //注意此字符串必须是真实路径，就是带包名的类路径，包名.类名
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}

3、利用class类获得实例化对象

try {
    //获得class对象
    Class<?> clazz = Class.forName("com.blog.reflect.Student");

    //1、对于无参构造方法
    //使用class的newInstance()方法实例化对象
    @SuppressWarnings("unused")
    Student student1 = (Student)clazz.newInstance();

    //2、对于有参构造方法使用Class对象的getConstructor()或getConstructors()方法
    //该方法返回一个来自于java.lang.reflect的Constructor对象,在调用getConstructor()的时候
    //应该保证构造方法和getConstructor()方法中的参数类型为其对应类型的包装对象

        //2-1、取得其中一个构造方法
    Constructor<?> con = clazz.getConstructor(String.class, Integer.class);
    Student student2 = (Student)con.newInstance("张三", 18);
    System.out.println(student2.name);


        //2-2、取得全部构造方法
    Constructor<?>[] cons = clazz.getConstructors();
    for (int i = 0; i < cons.length; i++) {
        System.out.println(cons);
    }
//注：getConstructor()和getConstructors()方法内的参数只能取得公有的构造方法
//并且getConstructor()方法的参数类型,顺序应该与对应类中构造方法的参数类型,顺序保持一致

    //3、对于访问权限为私有的构造方法使用getDeclaredConstructor()或getDeclaredConstructors()。
    //在访问时注意设置setAccessible(true);使其变为可访问的
    Constructor<?> dcon = 
    clazz.getDeclaredConstructor(String.class, Boolean.class, Integer.class);
    dcon.setAccessible(true);
    Student student3 = (Student)dcon.newInstance("李四", false, 18);
    System.out.println(student3.getAge());

  通过上述，我们可以知道：即使在我们不知道类的结构，不导入相应的包和类的情况下，也可以进行类的实例化操作

  值得一提的是：如果使用newInstance实例化对象，必须要求类中存在无参构造方法，因为在默认情况下，class类的newInstance()只能够找到无参，否则请使用getDeclaredConstructor()或getConstructor()方法

4、利用反射取得类中的方法

try {
    //得到class类对象
    Class<?> clazz = Class.forName("com.blog.reflect.Student");
    //调用构造方法
    Constructor<?> con = clazz.getConstructor(String.class, Integer.class);
    //给构造方法赋值
    Student student = (Student)con.newInstance("张三", 18);

    //getMethod(方法名, 方法参数类型)或getMethods()得到所有的普通方法(包括父类的方法)
    //但是无法得到访问权限为私有的方法

    //getMethod(方法名, 方法参数类型)
    Method m1 = clazz.getMethod("eat", String.class);
    //使用method.invoke(对象名, 方法参数支持多个参数)调用指定方法
    m1.invoke(student, "橘子");

    //getMethods()
    Method m2[] = clazz.getMethods();
    for(Method method : m2) {
        System.out.println(method.getName());
    }
    //getDeclaredMethod(方法名, 方法参数类型)或getDeclaredMethods()
    //可以得到访问权限为私有的方法,但是父类中的方法无法获取

    //getDeclaredMethod(方法名, 方法参数类型)
    Method m3 = clazz.getDeclaredMethod("writeDiary", String.class);
    m3.setAccessible(true);
    m3.invoke(student, "hallo word");

    //getDeclaredMethods()
    Method m4[] = clazz.getDeclaredMethods();
    for(Method method : m4) {
        System.out.println(method.getName());
    }
    } catch (ClassNotFoundException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (NoSuchMethodException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (SecurityException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (InstantiationException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (IllegalAccessException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (IllegalArgumentException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    } catch (InvocationTargetException e) {
    // TODO Auto-generated catch block
    e.printStackTrace();
    }

补充：

Method[] methods = cls.getDeclaredMethods();
for(Method method : methods) {
    //输出方法访问权限
    System.out.print(Modifier.toString(method.getModifiers()));
    //输出方法返回值
    System.out.print(" " + method.getReturnType().getSimpleName());
    //输出方法名
    System.out.print(" " + method.getName());
    String param = "";
    if (method.getParameterCount() != 0) {
        Class<?>[] params = method.getParameterTypes();
        for(int i=0; i<params.length; i++) {
            param += params[i].getSimpleName() + " args";
            if(i < params.length-1) {
                param += ",";
            }
        }
    }
    //输出方法参数
    System.out.print(" " + "("+ param +")");

    //获取方法包含的所有异常
    Class<?> exps[] = method.getExceptionTypes();
    if(exps.length > 0) {
        System.out.println(" throws ");
        for (int i = 0; i < exps.length; i++) {
            //输出异常
            System.out.println(" " + exps[i].getSimpleName());
            if (i < exps.length -1) {
                System.out.print(",");
            }
        }
    }
    System.out.println("{};");
}

5、利用反射调用类中的属性

  4-1取得所有继承而来的属性

    1. 取得指定属性：getField(String name) 

//得到class类对象
Class<?> clazz = Class.forName("com.blog.reflect.Student");
//调用构造方法
Student student = (Student)clazz.newInstance();
//得到name属性
Field f1 = clazz.getField("name");
//调用Field类中的set方法类似：setName()
f1.set(student, "张三");
//调用Field类中的get方法类似：getName()
System.out.println(f1.get(student));

    2. 取得所有属性：getFields() 

Field f2[] = clazz.getFields();
for(Field field : f2) {
    System.out.println(field.getName());
}

  4-2取得所有本类定义的属性

    1. 取得指定属性：getDeclaredField(String name) 

//得到class类对象
Class<?> clazz = Class.forName("com.blog.reflect.Student");
//调用构造方法
Student student = (Student)clazz.newInstance();
//得到私有的age属性
Field f3 = clazz.getDeclaredField("age");
//设置为可访问的
f3.setAccessible(true);
//调用Field类中的set方法类似：setAge()
f3.set(student, 18);
//调用Field类中的set方法类似：getAge()
System.out.println(f3.get(student));

    2. 取得所有属性：getDeclaredFields() 

Field f4[] = clazz.getDeclaredFields();
for(Field field : f4) {
    field.setAccessible(true);
    System.out.println(field.getName());
}

三、附：Student类内容如下：

package com.blog.reflect;

public class Student implements Person{
    public String name;
    protected Boolean sex;
    private int age;

    @SuppressWarnings("unused")
    private Student(String name, Boolean sex, Integer age) {
        super();
        this.name = name;
        this.sex = sex;
        this.age = age;
    }

    public Student(String name, Integer age) {
        super();
        this.name = name;
        this.age = age;
    }

    public Student(String name) {
        super();
        this.name = name;
    }

    public Student() {
        System.out.println("这是student的构造方法");
    }

    @SuppressWarnings("unused")
    private void writeDiary(String content) {
        System.out.println(this.name + "的日记内容" + content);
    }

    public void study(String subject) {
        System.out.println(this.name + "在学习" + subject);
    };

    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public boolean isSex() {
        return sex;
    }
    public void setSex(boolean sex) {
        this.sex = sex;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "student类";
    }

    @Override
    public void eat(String foot) {
        System.out.println(this.name + "在吃" + foot);
    }

}
